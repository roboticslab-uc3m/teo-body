Detailed Description

This controlboardwrapper2 subdevice exposes motor commands to CAN nodes modelled as wrapped motor raw subdevices (i.e. devices which implement the xxxRaw interface counterparts). At the core of this driver class, another set of wrapped devices implements hardware connection to a physical CAN bus, thus allowing CAN reads and writes that CanBusControlboard manages asynchronously with regard to exposed YARP commands (see CanReaderWriterThread).

This device also supports fake CAN buses and fake CAN nodes, see instructions.
